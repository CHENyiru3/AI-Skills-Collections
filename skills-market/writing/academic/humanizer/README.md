# Humanizer

`humanizer` is a writing cleanup skill for prose that sounds AI-generated,
robotic, generic, or unlike the intended voice.

It is designed as a final-pass editor:

- remove AI tells
- reduce corporate or over-smoothed phrasing
- restore natural rhythm
- preserve meaning and genre

This skill is not a default writing style overlay for every long response. It is
best used when the text already exists and needs voice calibration.

## Academic writing

For manuscripts, abstracts, methods, results, discussions, and reviewer
responses, use `academic-writing-editor` first. Then use `humanizer` as the
final anti-AI cleanup pass.
